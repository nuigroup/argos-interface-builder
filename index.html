Argos is a drag-and-drop GUI interface builder built in openFrameworks, dealing specifically with creating musically-oriented interfaces for multi-touch interaction. Argos is a project initiated by Dimitri Diakopoulos, with help from NUI Groupâ€™s Seth Sandler, as a Google Summer of Code 2009 project.

The initial code release includes ofxTouchAPI, a basic framework for creating multi-touch objects, and ofxArgosUI, an under-development UI toolkit consisting of a number of GUI controls with methods for both mouse and touch interaction. If you are interested in helping contribute, please get in touch! The development version of Argos will be released shortly on the SVN as a Visual Studio 2008 project.
